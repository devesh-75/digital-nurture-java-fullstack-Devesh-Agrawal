Parameter Passing via URL in React Router

Dynamic URL parameters allow passing data (such as IDs or query parameters) through the URL path.

1. Defining a Dynamic Route Path:
   - Use colon (:) prefix in the route path to define a dynamic parameter:
     <Route path="/trainer/:id" element={<TrainerDetail />} />

2. Extracting URL Parameters using useParams Hook:
   - In functional components, import and call useParams() hook from 'react-router-dom':
     import { useParams } from 'react-router-dom';
     
     export default function TrainerDetail() {
       const { id } = useParams();
       // id contains the parameter value passed via URL (e.g., "1")
     }

3. Navigating with Parameters:
   - Pass dynamic parameters via <Link> to attribute:
     <Link to={`/trainer/${trainer.TrainerId}`}>{trainer.Name}</Link>
